## V0.0.3 版本主题

围绕「个人 AI 知识库 + 记忆助手」做**多知识库治理、技能化、记忆智能化（反思/召回/跨会话/主动关心）、交互增强（划词/分享/群聊/语音）**，把单 Agent 助手升级为可定制、有记忆、能多角色协作的智能体平台。
